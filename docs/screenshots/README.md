# Placeholder для скриншотов

В этой папке должны быть размещены скриншоты приложения.

## Необходимые скриншоты

1. `homepage.png` - Главная страница приложения (1200x800px, оптимизированная)
2. `login.png` - Страница входа в систему (1200x800px)
3. `dashboard.png` - Дашборд после входа (1200x800px)
4. `mobile-view.png` - Мобильная версия приложения (375x667px)

## Требования к скриншотам

- Формат: PNG
- Качество: Высокое, без сжатия артефактов
- Размер: Оптимизированный для web (рекомендуется использоватьTinyPNG для сжатия)
- Стиль: Актуальная версия приложения в production окружении
- Контент: Демо-данные для демонстрации функциональности

## Инструкции по созданию скриншотов

1. Откройте приложение в браузере Chrome/Firefox
2. Используйте инструменты разработчика для эмуляции устройств
3. Убедитесь, что все элементы интерфейса загружены
4. Сделайте скриншот с помощью встроенных инструментов браузера или внешних инструментов
5. Оптимизируйте изображение для web
6. Сохраните с соответствующим именем файла

Пример команды для создания скриншота через Puppeteer:

```javascript
const puppeteer = require('puppeteer');

(async () => {
  const browser = await puppeteer.launch();
  const page = await browser.newPage();
  await page.goto('https://your-domain.com');
  await page.setViewport({ width: 1200, height: 800 });
  await page.screenshot({ path: 'docs/screenshots/homepage.png' });
  await browser.close();
})();
```
